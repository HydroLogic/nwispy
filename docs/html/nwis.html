Description:    
    Get information about existing parameters.
 
Argument(s):  
    match                   ->  regular expression match object
    
Return:
    code, description       ->  code and description of the parameter
 
i.e.:    
    string  ->  '#    06   00060     00003     Discharge, cubic feet per second (Mean)'
    
    groups  ->  ('#', '06', '00060', '00003', '     Discharge, cubic feet per second (Mean)')
    
    group(2) = '06'
    group(3) = '00060'
    group(4) = '00003'
    group(5) = 'Discharge, cubic feet per second (Mean)'
    
    return  ->  (code, description) -> '06_00060_00003', 'description: 'Discharge, cubic feet per second (Mean)'}
